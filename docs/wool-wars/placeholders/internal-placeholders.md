# Internal Placeholders

The following placeholders only work in the scoreboard and in some GUIs and messages.

### General Placeholders

| Placeholder            | Description                           |
|------------------------|---------------------------------------|
| `{date}`               | The current date                      |
| `{count_<group>}`      | Players in a specific group           |
| `{count_total}`        | Total players                         |
| `{wins}`               | The player's wins                     |
| `{losses}`             | The player's losses                   |
| `{games_played}`       | Number of games played                |
| `{kills}`              | The player's kills                    |
| `{deaths}`             | The player's deaths                   |
| `{wool_placed}`        | Amount of wool placed                 |
| `{blocks_broken}`      | Amount of blocks broken               |
| `{powerups_collected}` | Amount of powerups collected          |
| `{win_streak}`         | The player's win streak               |

### In-Game Placeholders

| Placeholder        | Description                                |
|--------------------|--------------------------------------------|
| `{time_formatted}` | The formatted game time                    |
| `{time}`           | The game time in seconds                   |
| `{id}`             | The arena's ID                             |
| `{displayname}`    | The arena's display name                   |
| `{round}`          | The current round                          |
| `{group}`          | The arena's group                          |
| `{state}`          | The arena's state                          |
| `{win_points}`     | The points needed to win                   |
| `{players}`        | The number of players in the arena         |
| `{max_players}`    | The maximum number of players in the arena |
